# Makey Makey

Can you use your [Makey Makeys](https://makeymakey.com/) to control Arcade games? Yes! Yes! Yes!

![A banana used to play a Arcade game with Makey Makey](/static/hardware/makey-makey/hands.jpg)

The Makey Makey keys just needs to be remapped to match [Arcade controllers keys](https://arcade.makecode.com/reference/controller).
Once remapped, you can use all the awesomeness of Makey Makey to build fun controllers for your games (up to 2 players!). 

## Remapping your Makey Makey

* go to https://makeymakey.com/pages/remap to start the remapping process
* change the keys so that it matches the following image

![Makey Makey remapping](/static/hardware/makey-makey/remap.png)

## Playing your games

You can play your Arcade games from the editor or simply by clicking **Share** and playing from the shared page. **Make sure to focus the game area so that it receives the keyboard commands**.

Have fun!
